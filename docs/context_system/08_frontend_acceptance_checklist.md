# 前端检查清单 - context_system

> Version: v0.1.0
> Last Updated: 2026-03-13
> Status: Draft

## 1. 功能检查

1. 文件列表可展示并支持分页/筛选。
2. 上传流程可完成，成功后列表可见新文件。
3. 可见性四种策略可配置并正确回显。
4. 下线/恢复按钮行为正确，状态显示同步更新。
5. 可触发重建索引并展示任务状态反馈。

## 2. 契约对齐检查

1. 所有请求字段与 `FC-*` 定义一致。
2. 枚举值使用后端原值，不做变体拼写。
3. 错误码分支覆盖主要失败场景。

## 3. 状态/异常/空态检查

1. 上传失败、权限失败、冲突失败都有明确提示。
2. 空列表时展示引导信息。
3. `SELECTED_USERS` 未选用户时阻止提交。

## 4. 回归检查

1. 不影响现有 pre-review 页面主流程。
2. 登录态过期后重定向/提示行为一致。
