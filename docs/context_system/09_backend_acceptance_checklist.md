# 后端检查清单 - context_system

> Version: v0.1.0
> Last Updated: 2026-03-13
> Status: Draft

## 1. 功能检查

1. 上传接口可创建文件记录与 ingest 任务。
2. 列表与详情查询按 ACL 和状态过滤。
3. 可见性更新可正确重写 ACL。
4. 下线/恢复后检索可见性立即生效。
5. 重建索引接口具备冲突检测与任务回写。

## 2. 契约对齐检查

1. `BC-*` 中路径、方法、字段与 FE 合同一致。
2. 枚举与错误码返回稳定。
3. `SELECTED_USERS` 校验规则符合合同定义。

## 3. 持久化与流程检查

1. 主记录与 ACL 更新事务一致。
2. 事件表完整记录关键变更操作。
3. 任务表状态转换满足 `PENDING->RUNNING->SUCCESS/FAILED`。

## 4. 错误与可观测性检查

1. 权限、参数、冲突、任务失败错误码正确。
2. 日志包含 `request_id/actor_user_id/file_id`。
3. 指标可观察上传成功率、重建成功率、ACL 拒绝次数。
